PROJECTS

Interactive Forest Health Dashboard

This project involved developing a web-based application to visualize forest health data. The front-end, built with HTML, CSS, and JavaScript, featured interactive maps and charts. The back-end, powered by Python, processed satellite imagery from Google Earth Engine to monitor deforestation, track canopy cover, and identify areas at risk of disease or fire. Users could filter data by region and time, providing a powerful tool for forest management and conservationists.

GIS-based Wildlife Migration Tracker

Aimed at wildlife researchers, this project created a sophisticated tool for tracking animal migration patterns. The system that integrates GPS data from collared animals with geospatial data processed in QGIS. The platform's back-end, written in Python, cleans and analyze the raw data, while the user-friendly front-end displays the migration routes on a dynamic map. This allowed/allows researchers to better understand animal behavior and plan conservation corridors.

Automated Land-Use Change Detection System

This project focused on automating the process of monitoring land-use changes over time. Using Google Earth Engine's powerful cloud-based processing capabilities, I developed a Python script to analyze historical satellite imagery and detect changes in land cover, such as the conversion of farmland to urban areas or wetlands to fields.
